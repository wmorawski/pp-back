## Installation

```bash
$ npm install
```

## Useful commands

```bash
$ npm install -g prisma
$ npm install -g graphql-cli
$ prisma init
$ prisma deploy
$ graphql get-schema --project database
$ graphql codegen --project database
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
